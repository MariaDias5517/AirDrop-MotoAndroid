# AirDrop-MotoAndroid
Aplicativo Android para transferência rápida e segura de arquivos offline entre dispositivos Motorola, usando Wi-Fi Direct, Bluetooth e NFC.
## Funcionalidades

- Transferência via Wi-Fi Direct para alta velocidade  
- Fallback automático para Bluetooth  
- Pareamento rápido via NFC  
- Interface simples, intuitiva e responsiva  

---

## Tecnologias utilizadas

- Kotlin  
- Android SDK (Wi-Fi Direct, Bluetooth, NFC APIs)
